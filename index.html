About

I recently graduated from the Web Development Immersive program at General Assembly where I studied full-stack JavaScript technologies such as Angular, Node, and jQuery, along with HTML and CSS. I pivoted from my fine arts background into web development after witnessing the impact of the internet on the fine art world. I am finding that skills that were valuable to me as an artist, such as process-oriented creative thinking, meticulousness, and the drive to see a project through from start to finish, are also valuable in the world of programming. I am now seeking a role where I can leverage my creative and technical skill sets within an organization that fosters continuous learning and growth.

Technologies, languages, frameworks and libraries:

  • JavaScript, jQuery
  • HTML, CSS, and the DOM
  • MVC, Routers and the Client-Server Model
  • MongoDB, Express, Node.js and Angular.js
  • Git, GitHub
  • Responsive Web Design and Bootstrap
  • API's, AJAX, JSON and Postman

librys

A handy reading app to help you track books you've read!
librys uses the Google Books API to allow users to easily search for books they've read. Books can be stored on their "bookshelf" for easy reference and can also be rated.

Technologies Used: JavaScript, Node.js, MongoDB, Express, Angular.js, UI Bootstrap, HTML, CSS

Access by clicking here or to install the app, fork the GitHub repository here. You will then need to:

1) Clone the forked repo to your local workspace

2) In your local app directory, use npm install to install app dependencies specified in package.json

3) Connect your local app to a local or remote database of your own
